# Unsaid Frontend

## Technologies Used

This project utilizes the following technologies:

### Tailwind CSS
[Tailwind CSS](https://tailwindcss.com/) is used for utility-first CSS styling, allowing for rapid UI development with pre-built classes.

### Material UI
[Material UI](https://mui.com/) provides React components that implement Google's Material Design, offering pre-designed components for faster development.

### Key Dependencies

- **React** (v19.0.0) - Frontend JavaScript library for building user interfaces
- **React DOM** (v19.0.0) - React package for DOM rendering
- **Material UI** (v7.0.1) - React component library implementing Google's Material Design
    - @mui/material - Core Material UI components
    - @mui/icons-material - Material Design icons for React
    - @emotion/react & @emotion/styled - Styling engine used by Material UI
- **Tailwind CSS** (v4.1.3) - Utility-first CSS framework

### Development Tools

- **Vite** (v6.2.0) - Next generation frontend tooling for fast development
- **ESLint** (v9.21.0) - Code quality and style checking
- **TypeScript Types** - Type definitions for React

## Getting Started

To run this project locally:

```bash
npm install
npm run dev
```
